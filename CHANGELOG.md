# 📋 Журнал изменений

Все важные изменения в этом проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Семантического версионирования](https://semver.org/lang/ru/).

## [0.2.0] - 2025-01-19

### 🎉 Добавлено

- Полная русская локализация Makefile с сохранением функциональности
- Современная структура проекта на основе UV + Ruff + Pyright + Pre-commit
- Комплексная система команд для разработки:
  - 📦 Установка и настройка (`setup`, `install`, `dev`)
  - ✅ Контроль качества кода (`lint`, `fix`, `check`, `security`)
  - 🧪 Тестирование (`test`, `coverage`)
  - 🐳 Docker поддержка (`docker-build`, `docker-run`)
  - 📚 Генерация документации (`docs`)
  - 🚀 Развёртывание (`release`, `deploy`)
- Красочный вывод с эмодзи и цветовой схемой
- Автоматические pre-commit хуки для качества кода
- Поддержка VS Code с настроенными расширениями

### 🔄 Изменено

- Обновлен README.md с современным дизайном и полной документацией
- Пересоздан GEMINI.md с инструкциями для Google Gemini AI
- Обновлен LICENSE с актуальной информацией

### 🛠️ Техническое

- Настроена современная Python разработка с:
  - UV для управления зависимостями и средой
  - Ruff для линтинга и форматирования
  - Pyright для проверки типов
  - Pre-commit для автоматических проверок
  - Commitizen для стандартизированных коммитов
- Интеграция с GitHub для CI/CD workflow
- Поддержка Docker для контейнеризации

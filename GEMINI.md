# GEMINI.md

Этот файл предоставляет руководство для Google Gemini AI при работе с кодом в этом репозитории.

## Обзор Проекта

MCPISIA - это современный Python-based MCP (Model Context Protocol) сервер с возможностями оркестрации GitHub агентов. Проект следует лучшим практикам 2025 года, использует Python 3.12+, FastAPI для асинхронных веб-сервисов и интегрируется с несколькими ИИ провайдерами, включая Google Gemini, Anthropic Claude и автоматизацию GitHub через PyGithub.

## Среда Разработки (Стек 2025)

### Современные Инструменты

- **UV** - Быстрый менеджер пакетов Python (заменяет pip/poetry)
- **Ruff** - Универсальный линтер/форматтер (заменяет Black, Flake8, isort, pyupgrade)
- **Pyright** - Быстрый проверщик типов от Microsoft (заменяет MyPy)
- **GitLeaks + Bandit** - Сканирование безопасности
- **Commitizen** - Conventional commits и семантические релизы
